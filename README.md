# 42sh (Shell project in 42 school)

## Installing

```
git clone https://github.com/krystalz42/42sh
cd 42sh
make
./42sh
```

## Content

1.	The shell support the following feature :
	-redirections:
```
">" ">>" "<" "<<" "<&" ">&" "&>"
```
	-logical operator:

```
"|" "||" "&&" 
```

### command separator

```
";" "&"
```
