# 42sh (Shell project in 42 school)

## Installing

```
git clone https://github.com/krystalz42/42sh
cd 42sh
make
./42sh
```

## Content

*The shell support the following feature *:

		_redirections :_

```
">" ">>" "<" "<<" "<&" ">&" "&>"
```

		_logical operator :_

```
"|" "||" "&&" 
```

		_command separator :_

```
";" "&"
```
